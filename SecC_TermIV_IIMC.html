↷ Rescheduled & Cancelled Classes

To reschedule a class: open any session by tapping it in Weekly or Calendar view, then tap ↷ Reschedule This Class. Changes propagate to the timetable, calendar, attendance tracker, and summary.

ℹ️ Cross-Device Sync Guide

Setup (one time, ~2 min):
1. Create a blank secret gist at gist.github.com
2. Generate a Personal Access Token with only the gist scope
3. Paste both above and click Test & Save

On another device:
Open this HTML file, go to Settings, enter the same token + gist ID, click Test & Save, then hit ⟳ Sync in the header.
